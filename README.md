Aero Weather Widget (JavaFX)
A desktop weather widget built with JavaFX for the Aero Dynamics brand.
It displays aviation-focused weather information with a clean UI and animated aircraft icon.
Features
Airport city input field
Temperature, sky condition, wind, and visibility display
Flight condition indicator
Animated aircraft (JavaFX TranslateTransition)
Refresh button enabled only when city is entered
